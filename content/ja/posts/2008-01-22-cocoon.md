Apache Cocoon を Mac OS X 上で動かす方法です。

<!--more-->

### 1. homebrew などを使って wget を入れておく。

```bash
brew install wget
```

### 2. JAVA_HOME のパスを通す。

```bash
cd ~
vim .bash_profile
export JAVA_HOME=/System/Library/Frameworks/JavaVM.framework/Versions/1.5.0/Home
source .bash_profile
```

### 3. cocoon ダウンロード

```bash
wget http://ftp.kddilabs.jp/infosystems/apache/cocoon/cocoon-2.1.11-src.tar.gz
tar xvfz cocoon-2.1.11-src.tar.gz
sudo mv cocoon-2.1.11-src /etc/cocoon
cd /etc/cocoon
sudo chmod 755 ./build.sh
sudo ./build.sh
sudo chmod 755 ./cocoon.sh
./cocoon
```
