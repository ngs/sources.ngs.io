[Sketch] の塗り色を [UIColor] / [NSColor] のイニシャライザー Swift コードとしてクリップボードにコピーするプラグイン [Copy Swift Code] を公開しました。

インストール方法
--------------

GitHub の [Releases] ページから最新版の Zip ファイルをダウンロード、解凍し、 `Copy Swift Code.sketchplugin` をダブルクリックして下さい。
